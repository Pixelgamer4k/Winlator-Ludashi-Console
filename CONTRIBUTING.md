Thanks for your interest in **Winlator Ludashi Console**.

## Before you start

This app is a **derivative** of Winlator. Please respect:

- [BrunoSX / Winlator](https://github.com/brunodev85/winlator)
- [Pipetto-crypto / Winlator Bionic](https://github.com/Pipetto-crypto/winlator)
- [StevenMXZ / Winlator-Ludashi](https://github.com/StevenMXZ/Winlator-Ludashi)

Read [CREDITS.md](CREDITS.md) and [NOTICE](NOTICE).

## Development

1. Use JDK 17 and Android SDK matching `app/build.gradle`
2. Run `./gradlew :app:assembleDebug`
3. Prefer small, focused PRs (Console UI vs native stack)

## Reporting bugs

Include device model, Android version, APK version (`0.1.0-beta`), and relevant logcat / Wine logs.
